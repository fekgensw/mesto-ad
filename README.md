# Mesto

Проект Mesto с интеграцией API, валидацией форм и сборкой на Vite.

## Ссылка на проект

- [GitHub Pages](https://YOUR_USERNAME.github.io/mesto-ad-deploy/)

## Команды

- `npm install` — установить зависимости
- `npm run dev` — запустить локальный сервер разработки
- `npm run build` — собрать проект в `dist`
- `npm run deploy` — опубликовать сборку на GitHub Pages
